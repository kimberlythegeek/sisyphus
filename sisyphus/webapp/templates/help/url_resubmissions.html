Date: The date and timestamp associated with the resubmitted URL.

signature: The custom signature or comments that a user supplied with the URL resubmission.

url: The URL that was submitted.

status: The status of the resubmitted jobs. There are three possible status values: Waiting, Executing, and Completed. Each status is displayed followed by the total number of jobs that have the associated status. Once all jobs are complete the status message is ALL JOBS COMPLETE.

Total Count: The total number of jobs associated with the URL.
